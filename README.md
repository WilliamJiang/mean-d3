
This is a MEAN-stack + D3 Application by William Jiang on 10/28/2015.

## Quick Start

```
$ git clone https://github.com/WilliamJiang/mean-d3.git
$ cd mean-d3
$ npm install
$ node server
```
then visit: 
```
http://localhost:3000/d3/example
```
that's it!


## Clean refresh

```bash
$ bower cache clean
$ npm cache clean
$ node server.js
```
